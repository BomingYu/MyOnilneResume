Projects

Random Number Generator Web

    Description
  • Random Number Generator is a fun and useful web application designed to provide users with the ability to generate custom random numbers. Users can observe the dynamic effect of random numbers beating on the page by setting the number and range of generated random numbers.
    Function
  • Users can input the number of random numbers to be generated in the setting area, and set the range of random numbers (0 to 100,000)
  • After clicking the "Run!" button, the generated random numbers will be continuously generated and show a beating effect in the display area.
  • After clicking the "Stop" button at any time, the random number generation stops, the page stops beating and the results are displayed.
    Technology
  • Use HTML5 to define the web page structure, including titles, buttons, and display areas, and build the basic framework of web pages.
  • Use JavaScript to write logic to implement the function of generating random numbers, handle user input, and control the generation and stop of random numbers. Through event monitoring and DOM operation, the interactive function of the button is realized.
  • The interface design is implemented with CSS3, including color schemes, responsive layouts and font choices.

Decision Maker Web

    Description
  • This webpage is a simple decision aid that can help you make a choice when you are faced with a choice and undecided. This little assistant can provide you with a little help.

  • Feel free to add as many different options as you want before making a decision.
  • Click “Make a Decision For Me Now!” button to make a choice for you and display it in the results area.
    Technology
  • Implemented using HTML, CSS, and JavaScript.
  • JavaScript generates as many options as you want, and defines each option arbitrarily. Displays the corresponding text result in the result area.
  • CSS is used to style the interface, making it visually appealing. Responsive layout.
